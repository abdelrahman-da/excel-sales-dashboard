# Excel Sales Dashboard

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Data Visualization

## Project Overview

This project focuses on analyzing sales performance using Microsoft Excel. The dashboard was built using Pivot Tables, Pivot Charts, Slicers, KPI Cards, and Geographic Analysis to transform raw sales data into meaningful business insights.

The dashboard provides an interactive environment for exploring sales trends, customer behavior, product performance, profitability, and regional sales distribution.

## Business Objectives

- Monitor overall sales and profit performance.
- Analyze product and category performance.
- Identify top-performing customers.
- Track sales and profit trends over time.
- Evaluate geographic sales distribution.
- Support business decision-making through interactive reporting.

## Key KPIs

- Total Sales: $6.18M
- Total Profit: $1.61M

### Sales Performance Analysis
